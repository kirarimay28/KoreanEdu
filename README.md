# 임용 스터디 앱

임용고시 그룹 스터디를 위한 웹 앱입니다.

## 기술 스택

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** (스타일링)
- **lucide-react** (아이콘)
- **localStorage** (데이터 저장)

## 주요 기능

### 로그인 / 회원가입
- 아이디, 비밀번호, 나의 다짐 입력 후 가입
- 가입 정보는 브라우저 localStorage에 저장

### 스터디 탭
- **고전 문학**: 작품명 · 작가 · 기출연도 / 시적 화자 · 시적 대상 · 갈래 · 주제 · 답안 작성 / 상호 피드백
- **현대 문학**: 작품명 · 기출연도 / 답안 작성 · 모범 답안 확인 · 사고 과정 / 상호 피드백
- 본인 기록과 스터디원 기록을 날짜별로 확인

### 개인 공부 탭
- 과목 선택: 국교론 · 현대 문법 · 중세 문법 · 개론서 · 기타(직접 입력)
- 공부 커리큘럼 / 기출 파악 여부 (O · △ · X) / 피드백 / 공부 시간
- 당일 총 공부 시간 자동 합산

### 오늘의 반성 및 피드백 탭
- 부족했던 부분 / 보완 방향성 작성
- 하단에 본인의 다짐 표시

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
